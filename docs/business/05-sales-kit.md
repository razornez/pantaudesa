# PantauDesa Sales Kit

## Core offer

PantauDesa dapat dijual sebagai layanan:

> Custom dashboard transparansi dan monitoring data publik untuk organisasi, komunitas, media, atau instansi yang ingin membuat data kompleks lebih mudah dipahami publik.

## One-liner

Kami membantu organisasi mengubah data publik yang rumit menjadi dashboard transparansi yang mudah dipahami, mudah dibagikan, dan bisa ditindaklanjuti.

## Target buyers

### 1. NGO / komunitas advokasi

Pain points:
- Punya data, tetapi sulit dibaca masyarakat.
- Laporan masih berbentuk PDF/spreadsheet.
- Sulit menunjukkan prioritas isu secara visual.

Pitch:
- Kami bantu ubah data advokasi menjadi dashboard publik yang jelas dan menarik.

### 2. Media lokal / media data

Pain points:
- Butuh angle berita dari data.
- Sulit membandingkan wilayah.
- Butuh visual interaktif untuk story.

Pitch:
- Kami bantu media membuat data story interaktif yang mudah dipahami pembaca.

### 3. Pemerintah daerah / desa / program publik

Pain points:
- Transparansi sering hanya berupa file PDF.
- Warga sulit memahami laporan.
- Data tidak komunikatif.

Pitch:
- Kami bantu membuat kanal transparansi yang ramah warga dan mudah diperbarui.

### 4. Donor / program development

Pain points:
- Perlu monitoring multi-wilayah.
- Butuh laporan berkala.
- Butuh dashboard progress publik/internal.

Pitch:
- Kami bantu memantau progres program dengan dashboard yang jelas dan siap dipresentasikan.

## Common objections

### "Kami sudah punya laporan PDF."

PDF bagus untuk dokumentasi formal, tetapi kurang efektif untuk publik. Dashboard membantu warga membaca inti data dengan cepat dan menemukan hal penting tanpa membuka laporan panjang.

### "Data kami belum rapi."

Justru tahap awal bisa dimulai dari data yang ada. Kami bisa bantu mapping struktur data dan membuat versi dashboard MVP terlebih dahulu.

### "Belum ada budget besar."

Kita bisa mulai dari pilot kecil dengan satu dataset atau satu wilayah. Tujuannya membuktikan value terlebih dahulu.

### "Takut data disalahartikan."

Dashboard bisa dilengkapi disclaimer, metodologi, catatan konteks, dan bahasa netral agar data lebih aman dibaca publik.

## Closing line

Kita tidak perlu mulai dari sistem besar. Kita bisa mulai dari satu dashboard kecil yang membuktikan bahwa data publik bisa dibuat lebih jelas dan berguna.
